# Syntecxhub_Maze_Solver_AStar
# Maze Solver using A* Search Algorithm

## Project Description
This project implements a Maze Solver using the A* search algorithm.  
It generates a random maze and finds the shortest path from start to goal using heuristics.

## Features
- Random maze generation
- A* search algorithm implementation
- Manhattan distance heuristic
- Handles unreachable cases
- Visualization using Matplotlib

## Tech Stack
Python, NumPy, Matplotlib, Heapq

## How to Run
python maze_solver.py
